Leiningen for automating Clojure projects without setting your hair on fire

Leiningen is the easiest way to get started with Clojure. With a focus on project automation and straightforward configuration, it gets out of your way and lets you focus on your code.

(defproject leiningen.org "0.1.0-SNAPSHOT"
  :description "Generate static HTML for http://leiningen.org"
  :dependencies [[enlive "1.0.0"]
                 [cheshire "2.2.0"]
                 [org.markdownj/markdownj "0.3.0-1.0.2b4"]]
  :main leiningen.web)

This site is still a work in progress! Please visit the Github page for complete information while we work on finishing it up.

Install

Leiningen might already be available in your preferred package manager. If not, it's still easy to install:

  1. Download the script
  2. Place it on your $PATH (eg. ~/bin)
  3. Set it to be executable. (chmod 755 ~/bin/lein)
